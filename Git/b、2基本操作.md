#### git add

> 该文件添加到暂存区

`git add *.c`：**.c 结尾文件**

`git add README.md`：**README.md 文件**

`git add .`：**所有文件**

`git add [file1] [file2] ...`：**file1、file2 ...**



#### git status

> 查看在你上次add之后是否有对文件进行再次修改

`git status`
